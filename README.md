Traffic Overhaul: An overhaul for how traffic spawns in GTA IV. 
===========

This is an attempt to overhaul how traffic is loaded and unloaded GTA IV. It loads cheap, mid-tier, luxury, and/or industrial vehicles depending on what's appropriate for the current neighborhood, and unloads inappropriate vehicles, on the fly rather than updating every few minutes like the vanilla game. It also spawns buses and delivery vans in every neighborhood.

The source code is specifically based on TrafficOverhaulEFLC.asi.
